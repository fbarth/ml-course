\begin{Slide}{Definição de dados}
	
Dados são valores de variáveis quantitativas ou qualitativas que pertencem a um conjunto de itens.	

\newpage

Dados são valores de variáveis quantitativas ou qualitativas que pertencem a um \emph{conjunto de itens}.	

\begin{itemize}
	\item \emph{conjunto de itens}: objetos de interesse. 
\end{itemize}


\newpage

Dados são valores de \emph{variáveis} quantitativas ou qualitativas que pertencem a um conjunto de itens.	

\begin{itemize}
	\item \emph{variáveis}: uma medida ou uma característica de um item.
\end{itemize}

\newpage

Dados são valores de variáveis \emph{quantitativas} ou \emph{qualitativas} que pertencem a um conjunto de itens.	

\begin{itemize}
	\item \emph{qualitativas}: cidade de origem, sexo, fez ou não tratamento.
	\item \emph{quantitativas}: peso, altura, pressão do sangue.
\end{itemize}

\end{Slide}


\begin{Slide}{Caracterização dos dados}
\begin{itemize}
\item A escala define as operações que podem ser realizadas sobre os
  valores do atributo. 
\item Em relação à escala, os atributos podem ser classificados como
  \emph{nominais}, \emph{ordinais}, \emph{discreto} e
  \emph{contínuo}. 
\item Os dois primeiros são do tipo qualitativo e os dois últimos são
  quantitativos.

\newpage


\item Na escala \emph{nominal}, os valores são apenas \emph{nomes
    diferentes}, carregando a menor quantidade de informação
  possível. Não existe uma relação de ordem entre seus valores.

\item Os valores em uma escala \emph{ordinal} refletem também uma
  ordem das categorias representadas. Dessa forma, além dos operadores
  de igualdade e desigualdade, operadores como $<$, $>$, $\ge$, $\le$ podem ser utilizados.

\newpage

\item Uma variável quantitativa que pode assumir, teoricamente,
  qualquer valor entre dois limites recebe o nome de \emph{variável contínua}.

\item Uma variável que só pode assumir valores pertencentes a um
  conjunto enumerável recebe o nome de \emph{variável discreta}.

\end{itemize}
\end{Slide}

\begin{Slide}{Exploração de dados}
Uma das formas mais simples de explorar um conjunto de dados é a
extração de medidas de uma área da estatística denominada \emph{estatística
descritiva}. A estatística descritiva resume de forma quantitativa as
principais características de um conjunto de dados.

\newpage

Tais características podem ser:

\begin{itemize}
\item Frequência;
\item Localização ou tendência central (por exemplo, a média);
\item Dispersão ou espalhamento (por exemplo, o desvio padrão);
\item Distribuição ou formato.
\end{itemize}

\newpage

No R é trivial identificar a média e mediana de um dado conjunto de valores 
para um atributo qualquer, como apresentado abaixo:

\begin{alltt}
mean(survey$Pulse)
median(survey$Pulse)
\end{alltt}

Ou sumarizar todos estes valores através de um único comando:

\begin{alltt}
summary(survey$Pulse)
\end{alltt}

\newpage

Além das informações textuais obtidas por

\begin{alltt}
summary(iris$Sepal.Width)
\end{alltt}

É possível obter um resumo visual da centralidade dos dados através do gráfico
\textit{boxplot}. No R é simples gerar este tipo de gráfico.

\small
\begin{alltt}
par(mfrow=c(1,2))
boxplot(iris$Sepal.Width, 
        outline= FALSE, main="Boxplot",
        xlab="Sepal Width")
boxplot(iris$Sepal.Width, 
        main="Boxplot modificado", 
        xlab="Sepal Width")
\end{alltt}

\end{Slide}

O segundo gráfico ilustra uma variação do gráfico \textit{boxplot}, conhecida como

1 menor que 1o quartil - 1,5 x (3o quartil - 1o quartil).
\end{Slide}


\begin{Slide}{Espalhamento de valores}
As medidas mais utilizadas para avaliar o \emph{espalhamento} de
valores é a \emph{variância} (var) e o \emph{desvio padrão} (sd). Sendo que o
desvio padrão é dado pela raiz quadrada da variância.

\newpage

Desvio padrão:
\begin{equation}
s = \sqrt{\frac{1}{n-1} \sum_{i=1}^{n}{(x_{i} - \overline{x})^{2}}}
\end{equation}

Variância:
\begin{equation}
s = \frac{1}{n-1} \sum_{i=1}^{n}{(x_{i} - \overline{x})^{2}}
\end{equation}

\end{Slide}

\begin{Slide}{Correlação}
Dados multivariados permitem análises da relação entre
  dois ou mais atributos. Por exemplo, para atributos quantitativos,
  pode-se utilizar uma medida de correlação para identificar a relação
  linear entre dois atributos.
\end{Slide}

\begin{Slide}{Coeficiente de correlação de Pearson}
Este coeficiente, normalmente representado por $\rho$ assume apenas
valores entre -1 e 1.

\begin{itemize}
\item $\rho = 1$ significa uma correlação perfeita positiva entre as
  duas variáveis. 
\item $\rho = -1$ significa uma correlação perfeita negativa entre as
  duas variáveis.
\item $\rho = 0$ significa que as duas variáveis não dependem
  linearmente uma da outra. No entanto, pode existir uma depência não
  linear. Assim, o resultado $\rho = 0$ deve ser investigado por
  outros meios.
\end{itemize}

\newpage

\begin{equation}
\rho = \frac{\sum_{i=1}^{n} (x_{i} - \bar{x})(y_{i} - \bar{y}) }
{\sqrt{\sum_{i=1}^{n} (x_{i} - \bar{x})^{2}} \times
  \sqrt{\sum_{i=1}^{n} (y_{i} - \bar{y})^{2}}}
\end{equation}

\begin{equation}
\rho = \frac{cov(X,Y)}{\sqrt{var(X) \times var(Y)}}
\end{equation}

\end{Slide}
