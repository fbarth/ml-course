\begin{Slide}{O que é Aprendizagem de Máquina?}

\begin{itemize}

\item Área de estudo que fornece aos computadores a habilidade de
  aprender sem serem explicitamente programados [Arthur Samuel
  (1959)].

\item Área de estudo que permite desenvolver sistemas computacionais sem explicitamente programá-los.   

\end{itemize}

\begin{Slide}{Exemplos de aprendizagem supervisionada}
\begin{itemize}

\item Estimar o preço de uma casa.
\begin{itemize}
\item atributos: tamanho, posição geográfica, material.
\item classe: preço (regressão).
\end{itemize}

\item Determinar se uma pessoa tem câncer benigno ou maligno.
\begin{itemize}
\item atributos: tamanho do tumor, formato do tumor, idade do paciente.
\item classe: tumor benigno ou tumor maligno (classificação).
\end{itemize}

\newpage

\item Determinar se é um texto publicado em uma rede social é
  inadequado ou não.
\begin{itemize}
\item atributos: quantidade de palavras encontradas no texto,
  quantidade de palavras proibidas encontradas no texto, quantidade de
  textos já criados pelo usuário, idade do usuário no sistema, quantidade de
  textos criados pelo usuário e moderados, ...
\item classe: texto adequado ou não (classificação).  

\item \textbf{classes}: texto adequado, texto inadequado, texto com propaganda (classificação com
  múltiplas classes).
\end{itemize}
\end{itemize}
\end{Slide}

\begin{Slide}{Exercícios}
Que problema deve ser tratado como problema de regressão e que
problema deve ser tratado como problema de classificação?
\begin{itemize}
\item A sua empresa possui 1.000 itens idênticos em estoque. Você quer
  predizer quantos destes itens serão vendidos nos próximos três meses.
\item Você quer examinar clientes seus e para cada um decidir se ele
  irá pagar todo o financiamento ou não.
\end{itemize}
\end{Slide}


\begin{Slide}{Exemplos de aprendizagem não supervisionada}
\begin{itemize}
\item Dado conjuntos de itens adquiridos na mesma compra, identificar padrões
  de compra.
\item Identificar padrões de navegação em sites.
\item Agrupar notícias semelhantes publicadas por várias fontes de
  informação.
\item Em uma rede social, identificar sub-grupo de pessoas.   
\end{itemize}
\end{Slide}

\begin{Slide}{Exercícios}
Quais dos problemas abaixo você iria resolver com uma abordagem não
supervisionada de aprendizagem?
\begin{itemize}
\item Dado e-mails rotulados como spam e não spam, desenvolver um
  filtro de spam.
\item Dado um conjunto de notícias encontradas na Internet, agrupá-las
  em conjunto de notícias que tratam do mesmo assunto.
\newpage
\item Dado uma base de clientes, descobrir segmentos de clientes.
\item Dado uma base de pacientes diagnosticados com diabetes ou não,
  aprender a classificar novos pacientes com diabetes ou não.
\end{itemize}
\end{Slide}
