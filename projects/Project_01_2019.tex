\section{Introdução}

O objetivo deste projeto é identificar quais clientes de uma instituição financeira irão executar uma determinada transação no futuro. Os dados disponíveis para este projeto são dados reais de uma instituição financeira. No entanto, por questões de sigilo, todos os dados sensíveis foram anonimizados.

\section{Método de trabalho}

Sugere-se seguir as seguintes etapas para a execução desta análise:

\begin{itemize}
\item Aquisição, análise exploratória e pré-processamento dos dados; 
\item Construção do modelo;
\item Análise do modelo, e;
\item Preparação do relatório e principais artefatos que devem ser
  entregues. 
\end{itemize}

\section{Itens que devem ser entregues}

Os seguintes itens devem ser entregues:

\begin{itemize}
\item Um relatório apresentando os principais resultados da
  análise. Este relatório deve ser entregue na forma de um Notebook Python.
\item O conteúdo do mesmo deve possuir os seguintes itens:
\begin{itemize}
\item introdução, com descrição do contexto e objetivo do trabalho;
\item descrição do método utilizado para alcançar o objetivo, e;
\item conclusão, ou seja, descrição dos resultados encontrados com a
  análise realizada. 
\end{itemize}
\item O relatório deve possuir duas figuras que ilustram e sumarizam
  bem os achados. Cada uma das figuras deve ter um \textit{caption}
  com uma descrição autocontida da figura.
\item Todos os comandos que fazem parte do script (aquisição, transformação, 
criação do modelo, avaliação, etc) devem estar presentes no Notebook Python. 
\end{itemize}
